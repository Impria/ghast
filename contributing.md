## Getting Started

1. Install dependencies

 * Git (<http://git-scm.com>)
 * Gulp (<http://gulpjs.com>)
 * Node.js (<http://nodejs.org>)
 * NPM (<http://npmjs.org>)
 * Sass (<http://sass-lang.com>)

2. Clone repository

   ```bash
   $ git clone https://github.com/impria/ghast.git
   $ cd carbon
   ```

3. Install node dependencies using npm

   ```bash
   $ npm install
   ```

4. Develop

   ```bash
   $ gulp watch
   ```
